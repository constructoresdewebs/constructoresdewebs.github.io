# Recursos gratuitos para crear tu presencia digital de forma profesional

Este repositorio reúne una colección de **plantillas web modernas, CV online, portfolios, componentes reutilizables y recursos de diseño** pensados para ayudar a cualquier persona a promocionarse en Internet sin necesidad de conocimientos avanzados de programación.

Mi objetivo es ofrecer herramientas accesibles, visuales y fáciles de personalizar para que puedas construir tu marca personal y mostrar tu talento al mundo.

**Nota:** Las plantillas han sido creadas con la ayuda de IA, pero cada una **ha sido revisada, ajustada y mejorada manualmente por mí** para garantizar un diseño cuidado y una estructura lista para usar.

## Qué incluirá este repositorio

- Plantillas completas de CV online listas para publicar
- Portfolios personales en diferentes estilos
- Ejemplos de landing pages para presentación profesional
- Recursos para mejorar tu marca personal
- Guías para publicar tu web en GitHub Pages
- Versiones con modo claro / modo oscuro
- Código limpio y fácil de editar (HTML, CSS, JS)

Cada plantilla está diseñada para ser:

- Totalmente responsive
- Moderna y visual
- Fácil de personalizar
- Gratuita y de código abierto

## Cómo usar una plantilla

1. Entra en la carpeta de la plantilla que te interese.
2. Descarga los archivos (index.html, styles.css, script.js).
3. Personaliza tu información, colores y contenido.
4. Sube tu proyecto a tu repositorio de GitHub.
5. Activa GitHub Pages para tener tu web online gratis.

## Modo claro / modo oscuro
Muchas plantillas incluyen un sistema de temas basado en variables CSS.
Puedes cambiar entre modo claro y oscuro con un solo botón, y el sitio recordará la preferencia del usuario.

## Publica tu CV online gratis con GitHub Pages
1. Crea un repositorio nuevo
2. Sube tu plantilla
3. Ve a Settings → Pages
4. Selecciona la rama main
5. Tu web estará disponible en una URL pública

Ideal para compartir en:

- LinkedIn
- Portales de empleo
- Redes sociales
